Download all the files in this folder on your raspberry pi.

Connect the DC-motor and the motor controlles as shown in:
https://www.electronicshub.org/controlling-a-dc-motor-with-raspberry-pi/

For the servo motor connect the brown wire to gnd, the red wire to 5V and the orange wire to pin 4.

Run both files to check if you have all the modules installed.

To use the full project first run the file: start_server.py
then run the MainMenu.py
